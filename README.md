# Aggregation: Metadata
Asynchronous metadata catalog writer for mytiki.com managed data catalogs. [AWS SQS](https://aws.amazon.com/sqs/) utilized for event triggering, queueing, and aggregation. [AWS Lambda](https://aws.amazon.com/lambda/) used for write execution.
